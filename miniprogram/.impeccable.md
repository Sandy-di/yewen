# .impeccable.md — 深空科技设计系统

> 智慧社区业主微信小程序 · 设计上下文文件
> 风格：深空科技 — 未来智慧社区
> 更新：2026-04-10

## 设计原则

1. **未来控制中心** — 每个界面都像精密的科技仪表盘，不是传统物业App
2. **暗色为底，发光为魂** — 深空蓝底色 + 电光青/极光紫发光交互
3. **数据面板美学** — 数字等宽发光，统计仪表盘式布局
4. **毛玻璃质感** — 卡片半透明模糊，不是纯白纸片
5. **克制发光** — 发光效果只在关键位置使用，不是处处霓虹
6. **对比度优先** — 暗色主题下文字可读性不可妥协

## 动效规范

- 入场：fadeIn + translateY，80ms递增延迟
- 全息扫光：hover态，0.6s ease
- 渐变漂移：8s循环，ease-in-out
- 脉冲发光：2s循环，ease-in-out
- 进度流动：1.5s循环，linear
- 交互反馈：scale(0.96) + glow增强，150ms

## 技术约束

- 微信小程序原生开发，不支持oklch/color-mix/容器查询
- backdrop-filter需2.9.0+基础库，必须做降级
- 动画仅用transform + opacity + box-shadow + background-position
- 不支持外部字体，仅系统字体栈
