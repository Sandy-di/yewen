# 智慧社区业主微信小程序

面向住宅小区业主的微信小程序，帮助业主和业委会完成具有法律效力的在线投票、实现物业财务公开、以及报修投诉的全流程跟踪。

## 🎯 核心功能

### 🔴 合规投票系统（MVP核心）
- 业委会发起投票，设置L1-L4身份核验等级
- 业主匿名投票，面积加权实时进度
- 区块链存证验证，结果公示+PDF下载
- 支持"双过半"和"双四分之三"表决规则

### 🔴 报修/投诉系统（MVP核心）
- 报修表单（分类+描述+照片）
- 物业抢单/派单，维修进度跟踪
- 业主验收评价，复修申请
- SLA超时自动升级

### 🟡 财务公示
- 月度收支报表提交
- 业委会审批流程
- 收支明细查询和图表可视化

### 🟡 社区公告
- 业委会公告、物业通知
- 置顶分类、已读统计

## 🔐 三角色体系

| 角色 | 权限 |
|------|------|
| 🏠 业主 | 查看投票、参与投票、发起报修、查看公示 |
| 🔧 物业 | 接收工单、更新状态、上报财务 |
| 📋 业委会 | 发起投票、审批财务、发布公告 |

## 🚀 使用方法

1. 使用**微信开发者工具**打开 `miniprogram/` 目录
2. AppID填写：测试可使用"测试号"
3. 编译预览即可

### 替换图标
当前TabBar图标为1x1透明占位符，发布前需替换为81x81px的设计图标：
```bash
cd assets/icons && node generate-icons.js
```

### 替换模拟数据为真实API
所有数据操作集中在 `utils/mock-data.js` 的 `mockApi` 对象中，后续替换为云函数调用只需修改该文件。

## 🎨 设计规范

- **主色调**：#0052D9（信任蓝）
- **辅助色**：#2BA471（成功绿）、#E34D59（警告红）、#ED7B2F（提醒橙）
- **文案风格**：亲切友好+透明可信
- **目标用户**：25-75岁业主群体，大字号高对比

## 📋 技术要点

- 微信小程序原生开发（WXML + WXSS + JS）
- 分包加载：主包+packageAdmin分包
- 前端模拟数据，预留API接口层
- CSS变量实现主题色管理
- 按需注入和懒加载优化
